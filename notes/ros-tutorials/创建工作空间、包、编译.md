# 创建工作空间、包、编译

## 1\.1 创建工作空间目录

`ros2_ws`是工作空间目录名称

```Plain Text
mkdir -p ~/ros2_ws/src
```

## 1\.2 进入工作空间

此处替换工作空间目录名称

```Plain Text
cd ~/ros2_ws
```

## 1\.3 克隆示例代码（推荐，可以快速了解目录结构 之后可以删除）

```Plain Text
git clone -b humble https://github.com/ros2/examples src/examples
```

此处会在ros\_ws下创建一个examples的包 如果不要可以直接删除

## 1\.4 创建自己的包



创建 Cpp\(CMake\)包  `cpp_pubsub`是包名

```Plain Text
ros2 pkg create --build-type ament_cmake cpp_pubsub
```

创建Python包  `python_pubsub`是包名

```Plain Text
ros2 pkg create --build-type ament_python python_pubsub
```

**如果没有指定，默认为ament\_cmake  **`my_package`是包名

```Plain Text
ros2 pkg create my_package
```

## 1\.5 编译

```Markdown
# 4.2 构建所有包
colcon build

# 4.3 构建特定包
colcon build --packages-select your_package_name

# 4.4 启用工作空间
source install/setup.bash

# 4.5 检查工作空间中的包
ros2 pkg list

# 4.6 清理构建文件
rm -rf build install log
```

在VSCode中 Ctrl\+Shift\+B可以发现里面有colon build 点击可以编译

## 1\.6 运行

ROS2的特点就是没有roscore 所以不用输出roscore

**记得source**

```Plain Text
source install/setup.bash
```

**ros2 run **

```Plain Text
ros2 run demo_nodes_cpp talker
```

**ros2 launch**

```Plain Text
ros2 launch demo_nodes_cpp launch_node.launch.py
```
