# 动态规划基础

动态规划（DP）用来解决**有重叠子问题和最优子结构**的问题。

## 核心思想

1. **状态定义**：用 `dp[i]` 或 `dp[i][j]` 表示某个子问题的解
2. **状态转移方程**：找到 `dp[i]` 与更小子问题之间的关系
3. **初始状态和边界**
4. **计算顺序**：自底向上 or 带备忘录的递归（自顶向下）

## 经典例题：爬楼梯

> 假设你正在爬楼梯，需要 n 阶到达顶部。每次可以爬 1 或 2 个台阶。有多少种不同方法？

**状态**：`dp[i]` 表示到达第 i 阶的方法数
**转移**：`dp[i] = dp[i-1] + dp[i-2]`
**边界**：`dp[1] = 1, dp[2] = 2`

```js
function climbStairs(n) {
  if (n <= 2) return n;
  let a = 1, b = 2;
  for (let i = 3; i <= n; i++) {
    [a, b] = [b, a + b];
  }
  return b;
}
```

## 常见 DP 题型

| 题型 | 例子 |
|------|------|
| 线性 DP | 爬楼梯、最大子序和 |
| 区间 DP | 矩阵链乘法、回文分割 |
| 背包 DP | 0-1 背包、完全背包 |
| 树形 DP | 二叉树直径 |
| 状压 DP | 旅行商问题（TSP） |

## 学习路径

- 先练会**线性 DP**
- 再学**背包问题**（0-1、完全、多重）
- 然后是**区间 DP**
- 最后**树形 DP** 和**状压 DP**
